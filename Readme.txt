-* jQuery-webFOCUS readme

- Sólo tienes que descargar la carpeta jQuery-webFOCUS, descomprimir si fuera necesario
  y abrir el index.htm con cualquier navegador*.

* El plugin está basado en cross-browsing, es decir, compatible con todos los navegadores. Aún así,
es recomendado utilizar el navegador IE(Internet Explorer) para que funcionen la parte de la generación de ficheros
ya que requiere la libreria ActiveXObjects, que sólo es soportada por éste navegador.
