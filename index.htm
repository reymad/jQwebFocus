Información general

Estas funciones utilizan el DOM de la página para generar estilos css de un informe FOCUS. Sólo hay que tener en cuenta dos cosas:

- El informe debe de guardarse en un HOLD FORMAT HTMLTABLE.

- El informe debe estar contenido en un div id="divInforme". (El nombre y el atributo es modificable).

Source: 

		 /* Código FOCUS */

		 TABLE FILE TABLA
		 PRINT *
		 ON TABLE HOLD AS HNAME FORMAT HTMTABLE
		 END
		 -RUN

		 /* Código HTML */

		 

		 	!IBI.FIL.HNAME;

*Por motivos de extensión, mostramos solo algunas de las funciones mas relevantes. El plugin contiene más funciones.

$('#divInforme').wfTdZebraClass(classOdd, classEven)

Esta función utiliza como selector el id="divInfome". Así de la siguiente manera, el informe contenido en #divInforme se cebreará con las clases que definamos y pasemos como parametros.

- "classOdd": serán las filas impares.(La clase se añadira a cada celda).

- "classEven": serán las filas pares.(La clase se añadira a cada celda).

Source: 

		
		  /* Clases CSS */
		  .oddRow{
		  	border: 1px solid silver;
		  	font-weight: bolder;
		  	background: lightgrey;
		  }
		  .evenRow{
		  	border: 1px solid silver;
		  	font-weight: bolder;
		  	background: deepskyblue;
		  }

		  /* HTML */

$.wfSideChar(cadena, qchar, charAtSides, charWhen)

Esta función resulta útil si vamos a pasar parámetros a un procedimiento fex, ya que el tratamiento del caracter ampersand (&) nos puede generar errores si no existe un espacio a cada lado de éste.La función pone un caracter a cada lado de un caracter especificado en una cadena, Sólo si al lado NO tiene un caracter especificado, por ejemplo un espacio. (Vale para cualquier caracter, que le pasemos como parametro).

P.ej: Tenemos en el input id="miInput" la siguiente cadena: "Johnson&Johnson".

Source: 

		/* Llamada a la función: $.wfSideChar( cadena, charAbuscar, charAponer, cuandoEsteChar ) */

$.wfChars(cadena)

Esta función convierte caracteres conflictivos p.ej.(&,€,+,%,#,etc)con procedimientos fex, tranformándolos en su correspondiente código ASCII. Útil para pasar url sin problemas de codificación.

P.ej: Tenemos la variable tipo String: "ESPAÑA"

Source: 

		  	var pais = "ESPAÑA";

		  	var pais2 = $.wfChars(pais);

		  	/* Ahora pais2 contiene la cadena 'ESPA%F1A' */

$.wfOnlyTheseChars(event, expReg)

Función para usar en eventos del tipo keyup, keypress. Cancela el evento si la tecla introducida NO se corresponde con las que pasamos como expresión regular.

P.ej: Sólo queremos que permita introducir letras, números y espacios en un input id="valida1".

$('#valida1').keypress(function(event){ $.wfOnlyTheseChars(event, "a-zA-Z0-9 ") });

Intenta introducir caracteres que no sean letras a-z, números o espacios.

Source: 

		  
		  $(document).ready(function(){
		  	
		  	$('#valida1').keypress(function(event){ $.wfOnlyTheseChars(event, "a-zA-Z0-9 ") });
		  
		  });

$.wfNotTheseChars(event, expReg)

Función para usar en eventos del tipo keyup, keypress. Cancela el evento si la tecla introducida SI se corresponde con las que pasamos como expresión regular.

P.ej: Sólo queremos que permita introducir cualquier caracter a excepción de letras, números y espacios en un input id="valida2".

$('#valida2').keypress(function(event){ $.wfNotTheseChars(event, "a-zA-Z0-9 ") });

Intenta introducir caracteres que sean letras a-z, números o espacios.

Source: 

		  
		   $(document).ready(function(){
		  	
		  	$('#valida1').keypress(function(event){ $.wfNotTheseChars(event, "a-zA-Z0-9 ") });
		  
		  });

$.wfGenerateMaster(nFields , nKeys, tName)

Esta función genera un fichero .mas (master de FOCUS). Lo genera en nuestro directorio en 'D:\\'.

- Se le pasan como parámetros en este orden: El número de campos, el número de claves, el nombre de la tabla (Se usará como nombre de fichero.).

- Si no se le pasan parámetros, genera una master basica de cinco campos, una clave y nombre wfMaster.mas.

- Se trata de una plantilla por lo que los nombres de campos y formatos de la master se tendrán que cumplimentar manualmente.

Source: 

		  /* Por ejemplo queremos una master de la tabla TABLAXXX de 10 campos, de los cuales 3 son clave.*/
		  
		  $.wfGenerateMaster(10, 3, 'TABLAXXX');

Ver master generada

$.wfGenerateTemplate(dir)

- Si no se le pasan parámetros, lo generá en 'D:\\'

Source:

		  $.wfGenerateTemplate();

Ver plantilla generada

$('#divInforme').wfGenerateFile(dir, nameFile, format, separator, fromRow)

Esta función utiliza el el DOM de la página para generar un fichero con el formato que le pasemos como parámetro, ya sea CSV, XLS, TXT, etc, a partir de un informe que hayamos guardado en un HOLD FORMAT HTMTABLE y contenido en un div con selector de jQuery. P.ej: div id="divInforme".

- Se le pasan cómo parametros (Opcionales): Directorio completo donde lo queremos, nombre del fichero, separador que queremos utilizar y desde que fila del informe queremos que genere el fichero.

- Si no se le pasan parámetros, lo generá en 'D:\\', con nombre 'NewFile', formato 'CSV', separador ';' y desde la fila 0 (la cabecera de columnas).

Source: 

		  /* Generar csv de informe abajo mostrado con separador ';' desde fila 0 (con cabecera) */
		  
		  $('#divInforme').wfGenerateFile();
		  
		  /* Generar txt de informe abajo mostrado con separador ',' desde fila 1 (omite cabecera) */
		  
		  $('#divInforme').wfGenerateFile('D:\\', 'miCSV', 'xls', ';', '1');
		 
		  /* Generar txt de informe abajo por defecto */
		  
		  $('#divInforme').wfGenerateFile('D:\\', 'miTXT', 'txt', ';', '1');
